\chapter{Resumen}
\cabeceraEspecial{Resumen}

Nuestro proyecto consiste en ampliar la herramienta PMCTrack, cuyo fin
es permitir la monitorización del rendimiento de un programa mediante el uso de
contadores hardware integrados en el procesador.

Esta ampliación está formada por tres nuevas características.
La primera consiste en modificar la parte interna de PMCTrack para incluir soporte de programas multihilo.
La segunda se trata de implementar una librería, *libpmctrack*, que provea de una API para que los desarroladores puedan monitorizar fragmentos de código haciendo uso de los contadores hardware.
La tercera es la construcción de una Interfaz Gráfica de Usuario o \emph{Graphical User
Interface}, que simplifica la configuración de eventos hardware y
permita visualizar gráficas de los datos obtenidos en tiempo real.

En este documento, además de la explicación detallada de cada una de estas características, hemos realizado diversos casos de estudio para comprobar y mostrar la utilidad de nuestra aportación a PMCTrack.
